Texto em destaque

Também podemos usar a tag MARK para destacar um texto com uma cor de fundo diferente.

Texto inserido ou deletado

Para indicar que temos um texto deletado, usamos a tag DEL.

Para indicar que um texto foi inserido, usamos a tag INS.

Para indicar que um texto é pequeno, usamos a tag SMALL.

Para indicar que um texto é sublinhado, usamos a tag U.

Texto especial

Para indicar que um texto é um código, usamos a tag CODE.

Para indicar que um 

texto é pré formatado
, usamos a tag PRE.

Para indicar que um texto

é uma citação em bloco
, usamos a tag BLOCKQUOTE.

Para indicar que um texto é uma citação curta, usamos a tag Q.

Para indicar que um texto é uma abreviação, usamos a tag ABBR.

Unir forças podemos utilizar ambas as tags em conjunto, para que tenham maior destaque
